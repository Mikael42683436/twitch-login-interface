# twitch-login-interface
Recriação da interface de login, cadastro e recuperação de conta da Twitch. Projeto desenvolvido para praticar manipulação de DOM com JavaScript, focado em acessibilidade e segurança de front-end.

# 🟣 Twitch Interface Clone (Structure)

Este projeto recria a estrutura das telas de autenticação da Twitch, focando em um HTML limpo, semântico e inteligente.

🚀 Diferenciais Técnicos no Código
* **Automação com JS: Implementação de loops `for` dentro do HTML para popular campos de data de nascimento de forma dinâmica.
* Gráficos Vetoriais: Uso de `SVG` para ícones e logotipos, garantindo performance e escalabilidade.
* Componentização: Estrutura organizada em containers (`login-container`) prontos para estilização CSS avançada.
* Foco em UX: Campos de formulário com sugestões de preenchimento (`autocomplete`) e tipos de entrada específicos para dispositivos móveis (`tel`, `email`).

 🛠️ Tecnologias
- HTML5
- JavaScript (DOM Injection)
- SVG
